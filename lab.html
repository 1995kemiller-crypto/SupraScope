The same moment. One feed an agent can reason with.

Every agent reads markets through flat numbers — price, volume, RSI. SupraScope turns the same moment into a structural narrative: memory layers, layer divergence, deviation fingerprints, topology events. Below, the same market moment is handed to an agent both ways, with the same question. A neutral referee — told nothing about which feed is which — judges whose reasoning is sharper.
