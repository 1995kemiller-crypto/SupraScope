Topology events — when cohesion breaks

Rupture
Two or more streams violently decohere simultaneously. Connector lines snap outward. Skin surface tears along the fault between layers.
Liquidation cascade · flash event · order book vacuum
Fold
One axis inverts sign. The object folds through itself. A layer that was positive goes deeply negative while others hold. Regime reversal forming.
Trend exhaustion · distribution · narrative flip
Collapse
All streams simultaneously compress toward center. The object shrinks to near its core. Connectors dim. Skin contracts to minimum. Eerie silence across all layers.
Liquidity void · pre-event stillness · participation withdrawal
Bloom
Sudden outward expansion across all layers simultaneously — distinct from normal expansion by its speed and universality. Every stream lights up together. Discovery energy.
Price discovery · new participant inflow · narrative ignition
Memory across all cohesion states
Each topology event leaves a structural fingerprint across all three memory layers. Short memory holds the immediate echo. Mid memory encodes the regime it occurred within. Long memory carries the constitutional imprint — so when the object approaches a configuration it has visited before, the gravitational pull of that prior state becomes part of the present geometry. The observer who has watched long enough will feel the rhyme before they can name it.
